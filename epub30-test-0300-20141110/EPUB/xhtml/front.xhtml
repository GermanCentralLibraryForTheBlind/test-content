About this Document

This document contains tests which are fundamental to accessibility of the Reading System for users with disabilities. The advanced tests are contained in a separate document.

The basic assumption of accessibility evaluation is that all Reading System should support reading by eyes, ears or fingers. It should be possible for users to read the content of this document by:

  • adjusting the display such as adjusting font size and colour combination,
  • reading the text with screen readers or TTS and
  • reading the text in a refreshable Braille display.

The Reading Systems should be evaluated using different combinations of assistive technology and all the tests given in this book should be performed. It is expected that an accessible Reading System will pass all the tests given in this book regardless of the input and output mediums used in testing. Testers may perform the tests given in this book using keyboard, mouse or touch/gestures as input and screen reader/self voicing, visual adjustments or Braille as output.

Conventions

The following conventions are used throughout the document:

All tests in this document are preceded by the label [FUNDAMENTAL] indicating that the test is fundamental to establish accessibility of the Reading System.

The test labels are followed by ID such as file--010. These test IDs correspond to the entries in the form on www.epubtest.org for accessibility evaluation.

Performing the tests

Each test includes a description of its purpose followed by the actual test statement, which can generally be evaluated to true or false. Some of the tests given in this book may not be applicable to the all Reading System and assistive technology configuration and in such cases the evaluation statement reads "Indicate Pass or Fail or Not Applicable".

The result of each test in this book should be recorded in the corresponding field in the form on the website and comments if any should be noted in the space provided in the online form.
